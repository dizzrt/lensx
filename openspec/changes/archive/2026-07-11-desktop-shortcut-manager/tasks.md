## 2. 启动器窗口动作

- [x] 2.1 新增启动器窗口动作路由，支持 `launcher.show`、`launcher.hide`、`launcher.toggle`。
- [x] 2.2 优化显示动作，按能力恢复窗口、展示窗口并请求焦点。
- [x] 2.3 优化隐藏动作，隐藏主窗口但保持应用进程常驻。
